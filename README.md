# zmq-chat-example
![ZMQ](0mq.png)
* SocketType:  REQ - ROUTER -DEALER -  PUB - SUB
* Environment: python 2.7
## Install lib requirement
```
$ pip install -r requirement.txt 
```
## Run
### BackSide
```
$ python server.py
```
```
$ python worker.py
```
### FrontSide
```
$ python client.py
```
